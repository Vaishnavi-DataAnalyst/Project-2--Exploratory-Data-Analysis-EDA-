# Project-2--Exploratory-Data-Analysis-EDA-

📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a cleaned E-Commerce Sales Dataset. The objective is to analyze customer purchasing behavior, sales performance, product trends, payment preferences, and order status patterns to uncover meaningful business insights.

🎯 Objectives
Understand the structure and characteristics of the dataset.
Perform descriptive statistical analysis.
Identify trends, patterns, and anomalies in sales data.
Detect potential outliers.
Analyze customer purchasing behavior.
Generate actionable business insights through visualizations.
